# Getting Started

This guide helps you get the Skilldiv.ai application running locally and explains where to find key documentation sections.

## Local preview

Install Jekyll (if not already installed) and serve the `docs/` folder:

```bash
gem install jekyll
jekyll serve --source docs --config docs/_config.yml
```

Then visit `http://127.0.0.1:4000` to preview the docs locally.
